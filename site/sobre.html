Sobre o projeto
Este aplicativo é o resultado de um projeto de TCC desenvolvido com dedicação pelo nosso grupo, composto por Thales, Otávio, Bárbara e Gabriel Henrique. Nosso objetivo foi criar uma solução prática e eficiente para o setor sucroenergético, focada no cálculo do faturamento de fornecedores de cana-de-açúcar.

Com uma interface intuitiva e funcionalidades voltadas para as necessidades específicas dessa área, o aplicativo simplifica e agiliza processos que antes demandavam tempo e esforço. Nosso compromisso é entregar uma ferramenta confiável, que contribua diretamente para a organização e o sucesso de quem atua nesse mercado.

Para tornar essa ideia realidade, utilizamos tecnologias modernas e confiáveis, como C# para o desenvolvimento da aplicação principal, SQL Server para o gerenciamento de dados e HTML, CSS e JavaScript para a criação do site de suporte ao aplicativo.

Nosso compromisso é entregar uma ferramenta intuitiva e funcional, que simplifique processos e contribua para o sucesso de quem atua nesse mercado.
